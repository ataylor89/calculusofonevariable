Problem 1 Let A = {1,5,7,10} and B={2,7,11,14}. Determine A ∪ B, A ∩ B, A \ B, B \ A, (A \ B) ∪ B, and (A \ B) ∪ (A ∩ B).

Solution 

A ∪ B = {1,2,5,7,10,11,14} 
A ∩ B = {7}
A \ B = {1,5,10}
B \ A = {2,11,14}
(A \ B) ∪ B = {1,2,5,7,10,11,14}
(A \ B) ∪ (A ∩ B) = {1,5,7,10}

Problem 3 Let S={1,2,3}. List all the members of S(S). If S is a set containing n elements, make a guess as to the number of elements in S(S).

Solution 

 
S(S) = {{}, {1}, {2}, {3}, {1, 2}, {1, 3}, {2, 3}, {1, 2, 3}}. S(S) has 8 members.
This is verified by Pascal's triangle. 1+3+3+1=8.
The cardinality of S(S) is (n, 0) + (n, 1) + (n, 2) + ... + (n, n) where n is the cardinality of S and (n, a) is n choose a.

Problem 4 Let S={1,2,3,4} be the universal set. Let A = {1,2,4}, B = {3,4} and C = {1, 2}. Determine each of the following sets: A ∩ B, B ∩ C, A ∪ B, A' ∩ B, (A' ∪ C) ∩ B.

Solution 

A ∩ B = {4}
B ∩ C = {}
A ∪ B = {1, 2, 3, 4}
A' ∩ B = {3}
(A' ∪ C) ∩ B = {3}

Problem 5 Illustrate the De Morgan law, (A union B)' = A' intersection B', via Venn diagrams

Solution 1 

De Morgan's law can be illustrated with a Venn diagram.
A' intersection B' is illustrated by the intersection of diagonal lines with opposite slope
The crossed diagonal lines do not intersect A union B
The crossed diagonal lines intersect the complement of A union B
The crossed diagonal lines with opposite slope illustrate A' intersection B'
The intersecting diagonal lines illustrate A' intersection B'

Problem 7 Explain what differences (if any) there are in the following sets: ∅, {∅}, S(∅), and S({∅}). Solution 

S(∅) = {A | A ⊂ ∅} = {∅} since the empty set is a subset of every set.
S({∅}) = {A | A ⊂ {∅}} = {∅, {∅}}
